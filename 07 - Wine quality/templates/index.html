API Documentation

Welcome to the API documentation for our Machine Learning application on Quality Wine.

This API will help you choose the best quality wine based on your preferences. Therefore, you will be enjoying your glass of wine for the evening.

Endpoint 1 : Enter data

GET / POST : https://my-wine-quality-app.herokuapp.com/

This endpoint allows you to make a prediction of quality wine with the characteristics that you will enter (ex: Alcohol, Chlorides ; float)

How to request this endpoint?

You just have to click on the button "submit" after having entered the data (11 features).
The submit button will use the POST method.
It will use the root ("https://my-wine-quality-app.herokuapp.com/")
The root actually applies the Machine Learning model trained.

Example: Input data and click on "Envoyer"

It returns a such message "The quality of this wine is : : 7.0 / 10." üëç üç∑

Endpoint 2 : API request

POST : https://my-wine-quality-app.herokuapp.com/predict

This endpoint allows you to make a prediction of quality wine with the characteristics that you send/post.

How to request this endpoint?

You should request this endpoint using a POST method with a JSON content. Your JSON should contain the 11 items required (ex: Alcohol, Chlorides ; float).

Example 1: curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[6.6,0.16,0.4,1.5,0.044,48,143,0.9912,3.54,0.52,12.4]]}' https://my-wine-quality-app.herokuapp.com/predict

It returns a JSON like this: 

          
  {
    'predict': 7.0
  }
